# 淼淼的PET备考游戏 - 运行指南

## 游戏简介
这是一个为淼淼准备的PET备考英语学习游戏，包含多种游戏模式：
1. 单词记忆卡片
2. 拼写游戏
3. 词义配对
4. 语法挑战

游戏基于完整的PET词汇表（1953个单词），分为23个主题。

## 运行方法

### 方法1：直接运行（最简单）
1. 在文件管理器中找到 `index.html`
2. 右键点击，选择"在浏览器中打开"
3. 或者直接双击打开

### 方法2：使用本地服务器
打开终端，运行以下命令：

```bash
# 切换到游戏目录
cd /Users/jimmychen/.openclaw/workspace/projects/pet-game-simple

# 启动Python简单HTTP服务器
python3 -m http.server 8080

# 启动后，在浏览器中访问：
# http://localhost:8080
```

### 方法3：使用Node.js（如果需要）
```bash
# 安装依赖（如果有）
npm install

# 运行服务器
npm start
```

## 游戏功能

### 1. 主界面
- 显示用户积分、等级、连续学习天数
- 四种游戏模式选择
- 今日词汇列表
- 成就展示

### 2. 游戏模式
- **单词记忆卡片**：翻转卡片记忆单词和释义
- **拼写游戏**：根据发音拼写正确单词
- **词义配对**：单词与释义配对
- **语法挑战**：选择题、填空题测试语法

### 3. 特性
- 进度追踪（掌握百分比）
- 成就系统
- 每日连续学习记录
- 单词朗读功能

## 技术架构
- HTML5 + CSS3 + JavaScript
- 响应式设计（支持移动设备）
- 本地存储（保存学习进度）

## 注意事项
1. 游戏进度会自动保存在浏览器的本地存储中
2. 如果遇到浏览器安全策略问题（如跨域），请使用方法2启动本地服务器
3. 游戏支持中英文界面，专为淼淼的学习需求定制
