# 🎮 淼淼的PET备考英语游戏

**专为淼淼（9岁）定制的英语学习游戏，随时随地玩！**

## ✨ 游戏特色

### 🎯 四大学习模式
- **单词记忆卡片**：翻转学习，记忆强化
- **拼写训练**：听说读写全面练习
- **词义配对**：快速反应，限时挑战
- **语法挑战**：选择题、填空题测试

### 📚 完整学习资源
- **1953个PET核心词汇**
- **23个主题分类**
- **剑桥大学官方词库**
- **中英双语支持**

### 🏆 激励系统
- **积分等级**：学习越多，等级越高
- **成就勋章**：完成挑战获得奖励
- **连续学习**：培养学习习惯
- **进度追踪**：可视化学习成果

## 🚀 快速部署

### 方案一：Vercel（推荐 ⭐）
**3分钟完成，全球访问，完全免费**

1. **注册Vercel**：[vercel.com](https://vercel.com)
2. **上传文件夹**：拖放本项目到Vercel
3. **获得网址**：如 `https://miao-pet-game.vercel.app`
4. **分享给淼淼**：扫码或直接访问

### 方案二：Netlify
**同样简单，免费托管**

1. 访问 [netlify.com](https://netlify.com)
2. 拖放`pet-game-simple`文件夹
3. 获得 `https://your-game.netlify.app`

### 方案三：GitHub Pages
**适合开发者的选择**


## 📱 iPad使用方法

### 方式一：扫码访问（部署后）
1. **生成二维码**：使用[QR Code Monkey](https://www.qrcode-monkey.com)
2. **iPad扫码**：相机对准二维码
3. **添加到主屏幕**：Safari → 分享 → 添加到主屏幕

### 方式二：直接输入网址
```

```

## 🔧 开发说明

### 技术栈
- **前端**：HTML5 + CSS3 + JavaScript
- **设计**：响应式布局，移动优先
- **存储**：浏览器本地存储
- **字体**：Google Fonts + Font Awesome

### 自定义修改
如需个性化调整：
1. **修改词汇**：编辑 `js/vocabulary.js`
2. **调整样式**：编辑 `css/style.css`
3. **添加功能**：在 `index.html` 中扩展

### 数据安全
- 所有学习数据存储在**浏览器本地**
- 云端只部署静态文件
- 不收集任何个人信息
