# 图片资源目录

## 图标说明

### 1. 游戏图标（用于添加到主屏幕）
建议创建以下图标：
- icon-192x192.png - PWA图标
- icon-512x512.png - PWA大图标
- favicon.ico - 浏览器图标

### 2. 游戏内图标
- word-icon.png - 单词记忆图标
- spelling-icon.png - 拼写图标
- match-icon.png - 配对图标
- grammar-icon.png - 语法图标

### 3. 成就图标
- medal-gold.png - 金牌
- medal-silver.png - 银牌
- medal-bronze.png - 铜牌
- crown.png - 皇冠

## 使用建议

### 在线图标替代方案
当前使用Font Awesome图标，如果网络问题可以使用：
1. 本地SVG图标
2. Emoji替代: ✏️📚🎮🏆

### 图标生成工具
1. https://favicon.io/ - 生成favicon
2. https://www.flaticon.com/ - 免费图标
3. https://icons8.com/ - 图标库

### 优化建议
- 使用WebP格式减小体积
- 图标不要超过100KB
- 考虑使用雪碧图减少请求
